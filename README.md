Projeto de Big Data em Python

Este projeto apresenta as etapas de limpeza, tratamento e organização de dados utilizando Pandas e PySpark.

A primeira parte do trabalho foi desenvolvida em Python com Pandas, onde foram realizadas limpeza dos dados, padronização de colunas, remoção de inconsistências e ajustes necessários para preparar o conjunto de dados.

Em seguida, PySpark foi utilizado para organizar e processar os dados em um fluxo adequado para Big Data, aplicando transformações adicionais e estruturando os dados para análise em escala.

Por último, foram gerados os gráficos e os requisitos do trabalho, incluindo regressão linear, distribuição de frequência, desvios padrão e médias.
